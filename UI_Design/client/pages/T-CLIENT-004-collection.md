## Sections Breakdown

### 1. Hero Banner
- **Type**: Full-width image or video
- **Height**: 60-80vh desktop, 50vh mobile
- **Content**: Collection name, tagline, CTA
- **Overlay**: Dark gradient for text readability

### 2. Editorial Content
- **Layout**: 2-column (image + text)
- **Purpose**: Brand storytelling
- **Mobile**: Stack vertically

### 3. Category Navigation
- **Type**: Image cards with overlay text
- **Links**: Filter shop by category within collection
- **Hover**: Scale effect on image

### 4. Product Grid
- **Layout**: 4 columns desktop, 2 mobile
- **Filtering**: Basic sort options
- **Loading**: "Load More" button
- **Products**: Collection-specific items

### 5. Cross-Sell Collections
- **Purpose**: Navigate to other collections
- **Layout**: 2 horizontal cards
- **Position**: Bottom of page


## Data Requirements
```typescript
interface CollectionPageData {
  collection: {
    id: string;
    name: string;
    slug: string;
    description: string;
    longDescription: string;
    heroImage: string;
    heroVideo?: string;
    featuredImage: string;
    categories: CollectionCategory[];
    productCount: number;
    startDate?: Date;
    endDate?: Date;
  };
  products: Product[];
  pagination: PaginationInfo;
  relatedCollections: Collection[];
}
```
