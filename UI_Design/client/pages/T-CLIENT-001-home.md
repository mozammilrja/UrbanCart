## Sections Breakdown

### 1. Hero Section
- **Height**: 80-100vh on desktop
- **Background**: Full-bleed image or video
- **Animation**: Subtle parallax or fade-in
- **CTA buttons**: Primary (filled) + Secondary (outline)

### 2. Announcement Bar
- **Type**: Scrolling marquee
- **Content**: Shipping offers, drop dates, promotions
- **Interaction**: Pause on hover

### 3. Featured Products Grid
- **Layout**: 4 columns desktop, 2 mobile
- **Cards**: Product image, brand, name, price
- **Hover**: Quick view button, wishlist icon

### 4. Category Grid
- **Layout**: 2x2 bento grid
- **Images**: Full-bleed category images
- **Overlay**: Text + CTA button

### 5. Collection Banner
- **Type**: Full-width promotional
- **Height**: 400-500px
- **Parallax**: Optional subtle effect

### 6. Bestsellers Carousel
- **Type**: Horizontal scroll/carousel
- **Arrows**: Desktop navigation
- **Indicators**: Dot pagination

### 7. Brand Values
- **Layout**: 3 columns with icons
- **Icons**: Custom or Lucide
- **Animation**: Fade in on scroll

### 8. Instagram Feed
- **Integration**: Instagram API or manual
- **Layout**: 6 square images
- **Hover**: Show likes/comments overlay

### 9. Newsletter
- **Layout**: Centered with CTA
- **Form**: Email input + submit
- **Validation**: Email format check

## Component Dependencies
- Header (shared)
- Footer (shared)
- ProductCard
- CollectionCard
- NewsletterForm
- AnnouncementBar
- ImageCarousel

## Data Requirements
```typescript
interface HomePageData {
  hero: HeroSlide[];
  announcements: string[];
  featuredProducts: Product[];
  categories: Category[];
  collectionBanner: {
    title: string;
    image: string;
    link: string;
  };
  bestsellers: Product[];
  instagramPosts: InstagramPost[];
}
```

## SEO Requirements
- H1: Hero headline
- Meta title: "APOSTLE | Premium Indian Streetwear"
- Meta description: Brand + key offerings
- Structured data: Organization, WebSite
- Canonical: https://apostle.in/

## Performance Considerations
- Hero image: LCP priority, preload
- Product images: Lazy load below fold
- Instagram: Load asynchronously
- Fonts: Preload critical fonts
