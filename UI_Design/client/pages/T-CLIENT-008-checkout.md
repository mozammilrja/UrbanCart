## Payment Methods (India)

### UPI
- Google Pay
- PhonePe
- Paytm
- BHIM
- Custom UPI ID

### Cards
- Credit Cards (Visa, Mastercard, Amex, RuPay)
- Debit Cards
- Save card option (PCI compliant)

### Net Banking
- Top banks listed
- Full bank list dropdown

### Wallets
- Paytm Wallet
- Amazon Pay
- Mobikwik

### Cash on Delivery
- Additional fee (₹50)
- Not available for all pincodes
- Order value limits

## Data Requirements
```typescript
interface CheckoutData {
  cart: CartData;
  addresses: Address[];
  selectedAddress: Address;
  deliveryOptions: DeliveryOption[];
  selectedDelivery: DeliveryOption;
  paymentMethods: PaymentMethod[];
  selectedPayment: PaymentMethod;
  coupon: Coupon | null;
  totals: OrderTotals;
}
```

## Security
- SSL/TLS encryption
- PCI DSS compliance for cards
- No card data stored on server
- Fraud detection integration
- OTP verification for high-value orders

## Component Dependencies
- StepIndicator
- AddressSelector
- AddressForm
- PaymentMethodSelector
- CardInput
- UPISelector
- OrderSummary
- ReviewOrder
