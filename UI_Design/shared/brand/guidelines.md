# UrbanCart Brand Guidelines

## Brand Philosophy

UrbanCart represents the pulse of Indian streetwear culture—bold, authentic, and unapologetically expressive. Our design language bridges traditional craftsmanship with contemporary urban aesthetics.

### Logo Specifications
- **Primary**: Black on white background
- **Inverse**: White on black background
- **Minimum Size**: 120px width (digital), 30mm (print)
- **Clear Space**: Height of "U" on all sides

### Logo Don'ts
- Don't stretch or distort
- Don't change colors outside brand palette
- Don't add effects (shadows, gradients)
- Don't place on busy backgrounds
- Don't rotate or tilt

## Typography

### Font Families

#### Outfit (Display & Headings)
```
A bold, geometric sans-serif for impact
```
- Weights: 600 (Semibold), 700 (Bold), 900 (Black)
- Usage: Headings, display text, hero sections

#### Inter (Body & UI)
```
A clean, highly legible sans-serif
```
- Weights: 400 (Regular), 500 (Medium), 600 (Semibold)
- Usage: Body text, buttons, navigation, forms

#### JetBrains Mono (Technical)
```
A developer-friendly monospace font
```
- Weight: 400 (Regular)
- Usage: Order IDs, codes, technical data

## Imagery

### Photography Style
- **High contrast**: Bold, dramatic lighting
- **Urban settings**: Streets, concrete, graffiti
- **Authentic**: Real people, candid moments
- **Diverse**: Representing all of India

### Product Photography
- Clean white/gray backgrounds for catalog
- Lifestyle shots for campaigns
- Model photography with attitude
- Detail shots showing quality

### Image Treatment
- No heavy filters
- Minimal retouching
- High saturation for lifestyle
- Natural skin tones preserved

## Iconography

### Style
- Line icons with 1.5-2px stroke
- Rounded corners (2px radius)
- 24x24px default size
- Consistent visual weight

### Icon Set
- Navigation: Home, Search, Cart, User, Menu
- Actions: Heart, Share, Close, Plus, Minus
- E-commerce: Cart, Bag, Truck, Package
- Status: Check, Alert, Info, Error

## Voice & Tone

### Brand Voice
- **Bold**: Confident, direct statements
- **Authentic**: Real, unfiltered expression
- **Inclusive**: Welcoming to all
- **Energetic**: Dynamic, action-oriented

### Writing Guidelines
- Use active voice
- Keep sentences punchy
- Speak directly to the customer
- Avoid corporate jargon
- Embrace cultural references

### Examples
```
❌ "We are pleased to inform you that your order has been shipped."
✓  "Your order is on its way! 🚀"

❌ "Please complete the checkout process to finalize your purchase."
✓  "Ready to check out? Let's go!"

❌ "Our latest collection features premium quality garments."
✓  "Fresh drops. Premium quality. Your style."
```

## Motion & Animation

### Principles
- **Purposeful**: Every animation has meaning
- **Quick**: Snappy, responsive feedback
- **Subtle**: Enhance, don't distract
- **Consistent**: Same patterns throughout

### Timing
| Type | Duration | Easing |
|------|----------|--------|
| Micro-interactions | 150-200ms | ease-out |
| Page transitions | 300ms | ease-in-out |
| Modals/Overlays | 200-300ms | ease-out |
| Loading states | Infinite | linear |

### Common Animations
- Hover lift: translateY(-2px)
- Button press: scale(0.98)
- Fade in: opacity 0→1
- Slide in: translateY(10px)→0

## Grid & Layout

### Container
- Max width: 1280px
- Padding: 16px (mobile), 24px (tablet), 32px (desktop)

### Grid System
- 4 columns (mobile)
- 8 columns (tablet)
- 12 columns (desktop)
- Gutter: 16px (mobile), 24px (tablet), 32px (desktop)

### Spacing Scale
Based on 4px unit: 4, 8, 12, 16, 24, 32, 48, 64, 96px
