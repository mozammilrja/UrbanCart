# UrbanCart

A modern e-commerce platform for urban shopping.

## Features

- Product catalog and search
- Shopping cart management
- User authentication
- Order processing
- Payment integration

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/UrbanCart.git

# Navigate to the project directory
cd UrbanCart

# Install dependencies
npm install

# Start the development server
npm run dev
```
